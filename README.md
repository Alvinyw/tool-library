# tool-library

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```
### Compiles and minifies for production
```
npm run build
```

### 注意事项
- Nodejs: 安装 v12.xx.xx 版本(若已安装，可以使用nvm切换nodeJs版本)
- Visual Studio Code: 代码开发工具(视个人喜好，推荐)
- GitLab: 开发协作管理工具
